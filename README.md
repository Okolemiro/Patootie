# Valentine Mini Project (GitHub Pages, ohne Build-Tools)

Dieses Mini-Projekt ist eine einzelne `index.html`, die direkt auf GitHub Pages läuft.

## Veröffentlichen auf GitHub Pages

1. **Neues GitHub-Repository erstellen**
   - Erstelle auf GitHub ein neues Repository (z. B. `valentine-page`).

2. **Dateien hinzufügen und committen**
   - Füge `index.html` und `README.md` ins Repository.
   - Committe und pushe die Dateien auf den Branch `main`.

3. **GitHub Pages aktivieren**
   - Gehe zu **Settings → Pages**.
   - Unter **Build and deployment**: **Deploy from a branch** auswählen.

4. **Branch und Ordner wählen**
   - Branch: `main`
   - Folder: `/ (root)`
   - Speichern.

5. **URL finden und testen**
   - Nach kurzer Zeit zeigt GitHub unter Pages die URL an, z. B.:
     `https://DEIN-USERNAME.github.io/DEIN-REPO/`
   - Öffne die URL und teste die Seite.

## Optional: Lokal testen

- Einfach `index.html` doppelklicken und im Browser öffnen, **oder**
- in VS Code mit der Erweiterung **Live Server** starten.
